-----------------------------------------------
pip - это установщик пакетов(инструментов)

Установка
ubuntu -> sudo apt install python3-pip
mac -> уже установлен

-------------------------------------------------

venv - это виртуальное окружение, куда устанавливаем пакеты

Установка
ubuntu -> sudo apt install python3-venv
mac -> brew install virtualenv

----------------------------------------------------

poetry - это установщик пакетов 

Установка
ubuntu -> sudo apt install python3-poetry
mac -> brew install poetry - project.toml(инфо о нашем проекте, "пакете")

1.poetry init - инициализирует нашу директорию как проект (создается файл - project.tom1)

2.poetry shell - он создает виртуальное окружение и активирует его, можно использовать 'poetrry env use python{вуерсия питона}'

3.poetry add name - установка пакета 'name', создается файл poetry.lock(подробная информация о пакетах и зависимостях, которые мы установили)

4.xargs poetry add < requirements.txt(файл с пакетами, которые установлены в проекте или нужно установить) - устанавливает пакеты из requirements.txt при помощи poetry

exit - для выхода из виртуального окружения
deactivate - для выхода из виртуального окружения
